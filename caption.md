สินค้าที่เปิดอยู่ในแท็บตอนนี้คือ product target ให้คุณใช้ข้อมูลชื่อสินค้า, วัสดุ, สี, ฟังก์ชัน และสไตล์จากหน้าเว็บนั้น
งานที่ต้องทำ:
1) เขียนจุดเด่นหรือข้อดี 4-5 ข้อ
- ใช้ Emoji แทนตัวเลขข้อ
- ข้อละไม่เกิน 5 คำ
- ตอบเป็น code block เดียว

2) เขียน short Thai captions สำหรับ product cover (Shopee/FB/IG)
- เกี่ยวข้องกับแคปชั่น
- เน้นคนเห็นแล้วต้องคลิก
- ให้ 5 ตัวเลือก
- ไม่เกิน 8 คำต่อแคปชั่น
- ตอบเป็น bullet list

3) เขียน hashtag สำหรับเพิ่มยอดขายของสินค้านี้บน Shopee/FB/IG
- ต้องเป็นคำค้นที่มีคนใช้จริง แนว generic search term
- ความยาวรวมอย่างน้อย 150-170 ตัวอักษร
- เรียงลำดับตามความเกี่ยวข้องมากที่สุดก่อน
- ตอบเป็น code block เดียว 1 บรรทัดต่อเนื่อง (ไม่มีขึ้นบรรทัดใหม่)

4) เขียน Clean URL
แปลง URL ยาวๆ เป็น format:

```
https://shopee.../product/{shop_id}/{item_id}
```

*(No Query Params)*
